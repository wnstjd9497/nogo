# 보톡스 상담 스크립트

> 사용 원칙
> - 환자 개인정보와 실제 케이스 정보는 넣지 않습니다.
> - 용량, 포인트 수 같은 수치 정보 대신 일반 원칙으로 설명합니다.
> - 쉬운 한국어로, 과장 없이, 신뢰감 있게 안내합니다.

## 1) 20초 오프닝(공감→문제정의)
- "표정 주름이나 턱·목 긴장 때문에 신경 쓰이셨죠. 충분히 불편할 수 있어요."
- "보톡스는 근육 움직임을 완화해 표정선이나 과긴장을 줄이는 데 도움을 줄 수 있습니다."
- "오늘은 고객님 얼굴 움직임을 기준으로, 어디를 어떻게 접근하면 자연스러운지 같이 정해볼게요."

## 2) 원인 3분류(환자 눈높이 버전)
- **근육 사용 요인**: "찡그리기, 눈 주위 힘주기, 이를 악무는 습관처럼 근육을 자주 쓰면 주름·긴장이 더 도드라질 수 있어요."
- **피부·노화 요인**: "시간이 지나며 피부 탄력이 떨어지면 같은 표정에서도 선이 더 진하게 남을 수 있어요."
- **생활·습관 요인**: "수면, 스트레스, 자세, 표정 습관이 겹치면 개선 속도와 유지감에 영향을 줍니다."

## 3) 선택지 2개(장단점 한 줄씩)
- **선택지 A: 자연스러운 표정 우선 접근**
  - 장점: "표정이 과하게 굳지 않도록 균형을 보기 좋습니다."
  - 단점: "처음엔 변화가 약하게 느껴질 수 있어요."
- **선택지 B: 체감 우선 접근**
  - 장점: "초기 변화 포인트를 더 빠르게 느끼기 쉬워요."
  - 단점: "개인에 따라 표정 어색함을 느낄 가능성이 있어 경과 확인이 더 중요합니다."

## 4) 추천 1개(왜 이걸 먼저 하는지)
- "고객님은 **자연스러운 표정 우선 접근**을 먼저 추천드립니다."
- "이유는 얼굴 전체 균형과 표정 습관을 먼저 맞춘 뒤, 필요하면 보강하는 방식이 만족도와 안전성 모두에 유리하기 때문입니다."

## 5) 기대치/일정(1차→재평가→보강)
- "1차에서 고객님이 가장 신경 쓰는 부위부터 기본 개선을 진행합니다."
- "재평가 때 표정 움직임, 주름 변화, 불편감 여부를 같이 확인합니다."
- "필요하면 보강을 결정하고, 목표는 '안 한 듯 자연스러운 변화'로 맞춥니다."

## 6) 흔한 반응(붓기/멍/통증 등) 안내 문장
- "시술 후에는 일시적인 붉어짐, 붓기, 멍, 뻐근함, 눌린 듯한 느낌이 있을 수 있어요."
- "대부분은 시간이 지나며 완화되지만, 개인차가 있어 회복 속도는 다를 수 있습니다."
- "표정이 낯설게 느껴지는 기간이 잠시 있을 수 있고, 경과를 보며 조정합니다."

## 7) 레드플래그(바로 연락해야 하는 증상) + 안내 문장
- 바로 연락이 필요한 경우:
  - "삼킴 불편, 발음 이상, 숨쉬기 불편처럼 기능 변화가 느껴지는 경우"
  - "눈꺼풀 처짐, 시야 불편, 심한 비대칭처럼 갑작스러운 변화가 나타나는 경우"
  - "통증·붓기·열감이 빠르게 심해지거나 전신 이상감이 생기는 경우"
- 안내 문장:
  - "이런 증상이 있으면 기다리지 말고 바로 연락 주세요. 빠르게 상태를 확인하고 필요한 조치를 안내드리겠습니다."

## 8) 사후관리(하지 말 것/하면 좋은 것)
- **하지 말 것**
  - "시술 부위를 강하게 문지르거나 압박하지 않기"
  - "당일 과음, 무리한 운동, 과도한 열 자극 피하기"
  - "안내받지 않은 추가 시술이나 제품을 임의로 쓰지 않기"
- **하면 좋은 것**
  - "하루 컨디션을 무리하지 않게 관리하고 충분히 쉬기"
  - "경과 기간 동안 불편한 점을 간단히 메모하기"
  - "안내받은 일정에 맞춰 재평가 받기"

## 9) FAQ 8개(“몇 번/바로 티/아픔/유지기간/부작용/다운타임/가격 질문 대응/다른 시술과 조합”)
1. **몇 번 해야 하나요?**
   - "정해진 횟수보다 고객님 반응을 보고 재평가해 결정하는 방식이 가장 정확합니다."
2. **바로 티 나나요?**
   - "초기 변화는 느낄 수 있지만, 자연스럽게 자리 잡는 과정까지 보는 것이 좋습니다."
3. **아픈가요?**
   - "불편감은 개인차가 있지만 보통 관리 가능한 범위입니다."
4. **유지기간은 얼마나 가나요?**
   - "개인 근육 사용 습관과 생활 패턴에 따라 달라서, 경과를 보며 유지 계획을 맞춥니다."
5. **부작용이 걱정돼요.**
   - "가능한 반응과 주의 신호를 먼저 설명드리고, 이상 징후는 빠르게 대응합니다."
6. **다운타임 있나요?**
   - "대부분 일상생활은 가능하지만, 일정이 중요하면 여유 있게 계획하시는 게 좋아요."
7. **가격이 왜 달라요?**
   - "부위 범위, 목표, 추적 관리 포함 여부에 따라 비용 구조가 달라질 수 있습니다."
8. **다른 시술과 같이 할 수 있나요?**
   - "가능한 조합은 있지만 고객님 우선 고민에 맞춰 순서를 정해 단계적으로 진행하는 편이 안전합니다."

## 10) 7문장 암기버전(핵심만)
1. "보톡스는 근육 움직임을 완화해 표정선과 긴장을 줄이는 데 도움을 줄 수 있습니다."
2. "고객님 표정 습관과 얼굴 균형을 먼저 보고 접근 방식을 정하는 것이 중요합니다."
3. "처음엔 자연스러운 표정 우선으로 시작하고 필요하면 보강하는 방식이 안전합니다."
4. "효과 평가는 바로 단정하지 않고, 자리 잡는 경과를 보며 판단합니다."
5. "붓기나 멍 같은 반응은 있을 수 있고, 대부분은 시간과 함께 완화됩니다."
6. "삼킴·시야·호흡 관련 이상이나 갑작스러운 비대칭은 바로 연락 주셔야 합니다."
7. "고객님 목표에 맞춘 단계적 계획이 만족도와 안전성에 가장 유리합니다."

|---|---|---|
| 미간 | 눈썹을 올리고 내리는 근육 균형, 눈꺼풀 주변 구조 | 눈꺼풀이 무겁게 느껴지거나 좌우 표정 차이가 생길 수 있어 경과 확인 필요 |
| 이마 | 전두근과 눈썹 위치의 균형 | 이마만 과하게 완화되면 눈썹이 무겁게 느껴질 수 있어 자연스러운 표정 목표를 먼저 합의 |
| 눈가(까치발) | 눈둘레근, 하안검 주변 얇은 조직 | 웃을 때 표정이 낯설게 느껴질 수 있어 과교정 피하고 단계적 접근 권장 |
| 콧등/토끼주름 | 코 주변 표정근과 인접 근육 | 웃을 때 코·입 주변 움직임 변화가 느껴질 수 있어 표정 변화 가능성 사전 안내 |
| 사각턱(저작근) | 저작근과 주변 씹기 기능 관련 구조 | 씹을 때 피로감, 질긴 음식 불편감이 일시적으로 있을 수 있어 식습관 조절 안내 |
| 턱끝(멘탈리스) | 턱끝 근육과 아랫입술 움직임 균형 | 말할 때 입 주변 어색함이 잠시 느껴질 수 있어 초기 경과 관찰 중요 |
| 입꼬리/입주변 | 구륜근·구각 관련 표정근 균형 | 발음·빨대 사용·입술 힘주기에서 낯선 느낌이 있을 수 있어 과도 교정 주의 |
| 목(플라티스마) | 목 앞쪽 표층 근육과 삼킴 관련 기능 | 목 당김감, 발성·삼킴 불편 가능성을 반드시 안내하고 이상 시 즉시 연락 교육 |
| 다한증(겨드랑이 등) | 피부 상태와 해당 부위 기능/위생 관리 | 시술 부위 압통·멍 가능, 땀 패턴 변화 체감 시점은 개인차가 있음을 설명 |
