# 기기 리프팅 환자선정 의사결정 트리 (HIFU vs RF vs 조합)

## 1) 입력 변수 정의
- **피부두께:** 얇음 / 보통 / 두꺼움
- **지방량:** 적음 / 보통 / 많음
- **처짐부위:** 턱선 / 볼 / 목
- **통증 민감도:** 높음 / 보통 / 낮음
- **다운타임 선호:** 매우 낮음(거의 없음 선호) / 보통 / 일부 허용

## 2) 빠른 추천 매트릭스(일반 원칙)

| 입력 패턴(핵심 조합) | 1차 추천 | 이유(요약) | 피해야 할 케이스 & 이유 |
|---|---|---|---|
| 얇은 피부 + 지방 적음 + 볼 처짐 | **RF 단독** 또는 **저강도 조합(RF 중심)** | 볼륨 소실 우려가 있어 피부질/탄력 중심 접근이 유리 | **고강도·과도한 HIFU 단독**: 매우 마른 얼굴에서 꺼짐/골격 도드라짐 위험 |
| 보통 피부 + 지방 보통 + 턱선 처짐 | **HIFU 단독** 또는 **HIFU+RF 조합** | SMAS/깊은층 타깃과 표층 탄력 개선을 함께 기대 | 통증 민감도 매우 높을 때 공격적 HIFU: 통증으로 시술 순응도 저하 |
| 두꺼운 피부 + 지방 많음 + 턱선/볼 경계 불명확 | **조합(HIFU+RF)** | 깊은층(윤곽) + 표층(탄력/결) 동시 개선 필요성이 큼 | 단일 모달리티에 과도한 기대: 개선 폭 제한 가능 |
| 목 처짐(피부 이완 중심) + 지방 적~보통 | **RF 단독** 또는 **조합(RF 우선)** | 목은 표층 탄력/피부결 요소 비중이 높아 RF 적합 | 고에너지 집중 시술 반복: 자극/회복 지연 위험 |
| 목 + 턱선 동반 처짐, 지방 보통 이상 | **조합(HIFU+RF)** | 턱선 윤곽 재정리(HIFU) + 목 피부탄력(RF) 분업 전략 | 통증/다운타임 요구 무시한 고강도 프로토콜: 중도 이탈 위험 |
| 통증 민감도 높음 + 다운타임 거의 불가(어떤 부위든) | **RF 단독(보수적 시작)** | 상대적으로 수용성 높은 방식으로 시작 후 단계적 증량 | 초기부터 강한 복합시술: 불편감·만족도 저하 가능 |
| 통증 민감도 낮음 + 다운타임 일부 허용 + 복합 노화소견 | **조합(HIFU+RF)** | 다층 접근으로 효율적 개선 가능 | 해부학 위험구역 평가 없이 강행: 합병증 위험 증가 |

## 3) 플로우(텍스트 의사결정 트리)

```text
[Start]
  ↓
(1) 지방량/볼륨 상태 확인
  ├─ 지방 적음 + 피부 얇음 ?
  │    ├─ Yes → RF 단독 또는 RF 중심 저강도 조합 고려
  │    │          (과도한 HIFU로 볼륨 감소/꺼짐 위험 주의)
  │    └─ No  → 다음 단계
  ↓
(2) 주 처짐 부위 확인
  ├─ 턱선 중심 ?
  │    ├─ Yes → HIFU 단독 또는 HIFU+RF 조합
  │    └─ No
  ├─ 볼 중심 ?
  │    ├─ 지방 많음/피부 두꺼움 → 조합(HIFU+RF) 우선
  │    └─ 지방 적음/피부 얇음 → RF 단독 우선
  └─ 목 중심 ?
       ├─ 피부 이완 위주 → RF 단독 또는 RF 우선 조합
       └─ 턱선 동반 처짐 큼 → 조합(HIFU+RF)
  ↓
(3) 통증 민감도·다운타임 선호로 보정
  ├─ 통증 민감도 높음 또는 다운타임 매우 낮음 선호
  │    → RF 단독/저강도·분할치료 우선
  └─ 수용 가능
       → 목표에 맞춰 HIFU 단독 또는 조합 유지
  ↓
(4) 최종 안전 체크
  ├─ 해부학 위험구역/금기/기저질환 확인
  ├─ 기대치 조율(즉시효과 vs 점진효과)
  └─ 병원 프로토콜 기반 최종 파라미터 결정
[End]
```

## 4) “피해야 할 케이스” 체크리스트(요약)

| 피하거나 보수적으로 접근할 상황 | 주의 이유 |
|---|---|
| 매우 마른 얼굴(지방 적음)에서 볼 부위 강한 HIFU 반복 | 추가 볼륨 감소, 꺼짐/해골화 인상, 만족도 저하 가능 |
| 통증 민감도가 높은데 고강도 HIFU를 장시간 단독 진행 | 통증으로 중단/불완전 시술 가능, 재내원 순응도 저하 |
| 다운타임 거의 불가 환자에게 초기부터 복합 고강도 시술 | 홍반/부종 체감으로 조기 불만·일정 지장 가능 |
| 목의 얇은 피부에 열·에너지 과집중 | 자극 반응/회복 지연 및 드물게 표재성 부작용 위험 |
| 해부학 위험구역 평가 없이 표준화되지 않은 벡터/샷 설계 | 신경·혈관 관련 합병증 위험 증가 |

---

### 한 줄 요약
- **마른 얼굴·얇은 피부·볼 중심 처짐:** RF 우선(볼륨 보존).
- **턱선 중심·중등도 처짐:** HIFU 또는 조합.
- **두꺼운 피부·지방 많음·복합 처짐:** 조합(HIFU+RF) 우선.
